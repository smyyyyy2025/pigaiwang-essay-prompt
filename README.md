# 📝 批改网英语作文高分 Prompt

本仓库提供一个**专为批改网设计的英语作文写作 Prompt**，遵循系统评分规则，帮助快速写出高分作文。

---

## 🎯 核心规则速览

| 规则 | 要求 |
|------|------|
| 段落结构 | 3段式（引言+主体+结论） |
| 句子数量 | ≥10句 |
| 单词数量 | 160-200词 |
| 句首连词 | **每个句子**必须有篇章连词开头 |
| 闪光句型 | 全文至少8个 |
| 闪光短语 | 全文至少12个 |
| 主题关键词 | 重复5-8次，不用代词替代 |
| 避坑 | 禁用 `people-centered` 等连字符词 |

---

## 🚀 使用方法

1. 打开 `批改网焚诀.md`
2. 阅读所有规则
3. 根据题目，套用规则写作
4. 或直接将 Prompt 输入 AI 生成作文
